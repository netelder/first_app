<a name="keys"></a>
## Keyboard Mappings ##

Keymap for Mac, to be nice, is *odd*, for example:

* **&#8984; P** is Parameter info (within method call arguments) not Print
* **&#8984; N** is Go to Class not New File 
* **^N** is New File not Next Line

To name just a few.
